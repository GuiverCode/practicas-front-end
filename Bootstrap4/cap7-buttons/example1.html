Button Styles

La clase .btn es la clase basica de un botón bootstrap.Luego estan las siguientes clases: btn-primary, btn-info, btn-success, btn-warning, btn-danger, btn-dark y btn-light.

Button Elements

Los elementos <a> y <input> también pueden ser usados como botones, agregando la clase .btn.

Link Button

Button Outline

Bootstrap 4 proporcina 8 botonoes outline. Ej: btn-outline-primary.

Block Level Button

La clase .btn-block crea un block button cuyo ancho se expande por todo el ancho de la ventana.

Active/Disable Buttons

La clase .active hace que un boton aparezca presionado, mientras que un boton .disabled no puede ser clicleable.
