Alert Messages

Bootstrap 4 proporciona mensajes de alerta.

Los mensajes de alerta son creados con la clase .alert, seguido de una de estas clases: alert-primary, alert-info, alert-secondary, alert-success, alert-warning, alert-danger, alert-dark, alert-light.

Primary! Esta caja de alerta indica una acción importante.
Success! Esta caja de alerta indica una acción exitosa.
Info! Esta caja de alerta contiene alguna información.
Warning! Esta caja de alerta contiene una advertencia.
Danger! Esta caja de alerta indica peligro o una potencial acción negativa.
Dark! Esta caja de alerta es de color gris.
Light! Esta caja de alerta es de color ligero.

Alert Links

Agregando la clase .alert-link cualquier elemento a dentro de la caja de alerta tendra un estilo acorde a la caja de alerta.

Closing Alerts

Para poder cerrar un alert message se debe agregar la clase .alert-dimissable y un elemento botón. Luego establecer los atributos del botón con los siguientes valores: class="close" y data-dismiss="alert".

Presione X para cerrar este mensaje de alerta.
Tip: Es recomendable usar &times en lugar de X.

Animated Alerts

Las clases .fade y .show agregan un efecto de desvanicimiento cuando se cierra el mensaje de alerta.

Presione X para cerrar este mensaje.
